Problem #1 Is Your Team The Sum of a Bunch of Squares?

It is fairly well known that the sum of the first N positive integers is N(N + 1) / 2. In other words, 1 + 2 + 3 + ... + (N-2) + (N-1) + N = N(N + 1)/2. Lesser known is that the sum of the first N perfect squares is N(N + 1)(2N + 1)/6. In other words, 1 + 4 + 9 + 16 + ... + N2 = N(N + 1)(2N + 1)/6.

INPUT- A positive integer N #< 1000, from file input1.

OUTPUT- The sum of the first N squares, to file output1.

EXAMPLE- Input: 5

Output: 55
